## Reason Code 10.4 — Other Fraud (Card-Absent Environment) [더미]

- **정의(요약):** 카드 비대면 환경에서 카드홀더가 인지·승인하지 않은 부정거래.
- **적용 조건(요약):**
  - 카드홀더가 거래를 본인이 하지 않았다고 진술
  - 카드사가 부정거래로 분류, 다른 사유코드(상품 미수령 등)로 처리 대상이 아님
- **필수서류:**
  - 부정거래 신고서(또는 카드홀더 미인지 진술서)
  - 문제 거래 내역
- **승인 가능성↑ 추가자료:** 배송지/청구지 불일치, IP·디바이스 불일치 로그
- **증빙 불가/주의:** 본인이 일부라도 이용을 인정하면 부정거래로 보기 어려움 → 13.x 검토

## Reason Code 13.1 — Merchandise / Services Not Received [더미] ⭐가이드 케이스

> 워크숍 가이드 케이스(`sample-data/case-13.1`)의 사유코드. 기본은 **인바운드(매입사 재반박)** 관점.

- **정의(요약):** 결제했으나 약속된 상품·서비스를 (약속일까지) 받지 못함.
- **적용 조건(요약):**
  - 카드홀더가 대금을 지불했고, 상품/서비스를 수령하지 못함
  - 가맹점에 해결을 시도했으나 미해결(환불 거절 등)
  - 분쟁 제기 시점에 약속된 제공일이 경과

### (A) 발급사 제출요건 — 아웃바운드/1CB [더미]
- **필수서류:**
  - 미수령 진술(무엇을, 언제까지 받기로 했는지)
  - 주문·결제 증빙(주문확인, 영수증 등)
  - 가맹점 연락/해결 시도 기록
- **승인 가능성↑ 추가자료:** 약속 배송일 경과를 보여주는 증빙, 가맹점의 환불 거절 회신

### (B) 매입사 응답요건 — 인바운드/재반박 (Table 11-94/11-95) [더미] ⭐기본
- **필수/유리(compelling) 증빙:**
  - 서비스·상품이 실제 제공·이용되었음을 입증하는 가맹점 자료
  - (항공 거래 예) 탑승확인서·passenger manifest·여정/운항 증빙
  - 거래 인증기록(예: 3DS 인증 결과), 거래 상세
  - 배송·수령 증빙(서명·전자배송 로그 등, 해당 시)
- **방어 논거 예:** 카드홀더가 실제로 서비스를 이용(탑승)했음이 입증되면 "미수령" 주장 반박 가능
- **주의:** 위 항목·테이블 번호는 더미. 실제 compelling evidence 요건은 실 규정(Table 11-94/11-95)으로 검증.

- **증빙 불가/주의(공통):** 부분 수령·하자 문제면 13.3 검토. 단순 변심은 대상 아님.

## Reason Code 13.3 — Not as Described or Defective Merchandise/Services [더미]

- **정의(요약):** 받은 상품·서비스가 설명과 다르거나 하자가 있음.
- **적용 조건(요약):**
  - 상품/서비스를 수령함
  - 받은 것이 가맹점 설명과 다르거나 결함이 있음
  - 가맹점에 반품·수정·해결을 시도했으나 미해결
- **필수서류:**
  - 하자/불일치 진술(설명 vs 실제 비교)
  - 상품 설명 자료(가맹점 광고·상세페이지 등)
  - 반품/해결 시도 기록
- **승인 가능성↑ 추가자료:** 하자 사진, 전문가 소견서, 가맹점 회신
- **증빙 불가/주의:** 아예 못 받았으면 13.1. 단순 변심·기대불일치는 대상 아님.
