## Reason Code 13.6 — Credit Not Processed (취소/환불 미이행) [더미] ⭐핵심

> 이 도구가 가장 집중하는 사유코드입니다(승소 사례·자료가 풍부). 정의·근거는 **§11.10.7 계열**.

- **정의(요약, §11.10.7.1):** 카드홀더가 **환불(credit) 또는 거래 무효(void) 영수증을 받았으나
  그 credit이 카드에 처리되지 않은** 경우, 발급사가 1CB(Dispute)를 개시.
  (ATM은 원거래가 취소·취소처리됐는데 입금조정 유효성을 다투는 경우 포함.)
- **적용 조건(요약):**
  - 거래영수증에 **"void" 또는 "cancelled" 표기**가 있어 환불·취소가 확정됐음 (§11.10.7.2 Dispute Rights)
  - 그럼에도 카드에 해당 credit이 입금되지 않음
  - **시간제한(§11.10.7.4 요약):** Credit 영수증 일자로부터 **15일 대기 후 개시, 120일 이내** 처리
    (미기재 영수증·540일 상한 등 예외 각주 존재 — 실 규정 확인)
- **발급사 필수서류 (§11.10.7.5):** 다음 중 **하나**
  - ① Credit Transaction Receipt 사본
  - ② **void/cancelled 표기**된 Transaction Receipt 사본 (예: 가맹점 화면의 VOID 캡처, 현지 매장 취소 영수증)
  - ③ 카드홀더에게 credit이 발생해야 함을 입증하는 기타 기록
  - 즉 핵심은 **"환불·취소가 확정됐다는 증거"** 입니다.
- **승인 가능성↑ 추가자료:**
  - 가맹점이 환불·취소를 확인해준 **연락기록**(채팅·메신저 스크린샷, 이메일 회신 등)
  - 그럼에도 카드에 입금되지 않았다는 **카드홀더의 반복 추적·미이행 진술**
  - 보증금·청구 명세(호텔 folio 등) 등 원거래·환불대상 식별 자료
- **무효/주의(§11.10.7.3 요약):** Mobile Push Payment · Straight Through Processing ·
  Visa Cash-Back의 캐시백 부분 · 무인주유기(AFD) 등은 대상 아님 → 실 규정으로 재확인.
  단순 변심·미수령은 13.6이 아님(미수령은 별도 검토).

## Reason Code 12.5 — Incorrect Amount (청구금액 상이) [더미]

> 13.6 대비 경량 요약. 근거는 **§11.9.4 계열**.

- **정의(요약, §11.9.4.1):** 거래 금액이 틀렸거나 가산·자릿수 전치(transposition) 오류가 난 경우.
  Dispute 금액은 **차액(difference)으로 한정**(§11.9.4.2). 시간제한 **120일**(§11.9.4.4).
- **발급사 필수서류 (§11.9.4.5 요약):**
  - **올바른 거래금액**이 적힌 Transaction Receipt 사본 또는 기타 기록
  - (ATM은 올바른 입금조정액 certification)
- **주의:** T&E 견적가-실제청구 차액, No-Show, 가맹점이 금액 변경 권리를 가진 거래 등은 무효 대상일 수 있음 → 실 규정 확인.

## Reason Code 12.6 — Duplicate Processing / Paid by Other Means (이중청구 / 타수단 결제) [더미]

> 13.6 대비 경량 요약. 근거는 **§11.9.5 계열**.

- **정의(요약, §11.9.5.1):**
  - **이중처리(Duplicate):** 동일 결제수단·동일 거래일·동일 금액으로 **단일 거래가 두 번 이상** 처리.
  - **타수단 결제(Paid by Other Means):** 동일 상품/서비스를 **다른 수단으로 이미 결제**.
  - 타수단 결제는 **가맹점과 먼저 해결 시도**한 뒤에야 개시 가능(§11.9.5.2). 시간제한 **120일**(§11.9.5.4).
- **발급사 필수서류 (§11.9.5.5 요약):**
  - **이중처리:** 유효 거래의 **일자 + ARN certification**
  - **타수단 결제:** 가맹점과 해결 시도했다는 certification + **타수단 결제 증거**(타카드/계좌 명세서, 현금영수증, 취소수표 앞뒤 사본 등)
  - "별개 거래 입증"용: 2장의 별도 거래영수증 또는 두 거래가 서로 다른 상품/서비스임을 입증하는 기록
- **주의:** 서로 다른 가맹점에 지불한 경우는 무효(§11.9.5.3). 경찰 신고서 등은 **정황 보강**일 뿐, 필수서류는 위 규정 항목 → 실 규정 확인.
